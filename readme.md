# FaceMatcher

FaceMatcher is a cutting-edge web application for real-time face detection and recognition, built with Node.js and face-api.js. It features a modern, glassmorphic UI designed for a premium user experience.

## ‚ú® Features

- **Real-time Face Detection**: High-performance face tracking using TinyFaceDetector.
- **Landmark Detection**: visualizes 68 facial landmarks.
- **Expression Analysis**: Detects emotions (happy, sad, angry, etc.) in real-time.
- **Modern UI**: Dark-themed, glassmorphic design with smooth animations.
- **Responsive**: Works seamlessly across devices.

## üöÄ Getting Started

### Prerequisites

- Node.js installed on your machine.

### Installation

1. Clone the repository or navigate to the project folder:
   ```bash
   cd "D:\Project\Web project\FaceMatcher"
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Ensure AI models are downloaded to `public/models`. (Completed during setup).

### Running the Application

Start the development server:

```bash
npm run dev
```

Visit `http://localhost:3000` in your browser.
